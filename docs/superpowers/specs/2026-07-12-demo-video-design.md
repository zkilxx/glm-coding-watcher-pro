# GLM Coding 监测助手演示视频设计

## 交付规格

- 时长：约 60 秒
- 画幅：16:9
- 分辨率：1920×1080
- 格式：MP4（H.264）
- 语言：简体中文字幕，无配音
- 声音：轻量背景音乐，音量不干扰字幕阅读
- 用途：GitHub README、B站、群聊和产品演示

## 视觉风格

- 延续插件的蓝白配色、圆角卡片和清晰信息层级。
- 整体采用高级科技产品发布片风格，以深蓝空间背景衬托真实界面。
- 使用真实插件截图和界面组件，不展示账号头像、验证码细节或支付信息。
- 通过平滑缩放、局部框选、光标移动和短字幕突出操作重点。
- 不伪造购买成功；“自动点击”使用流程示意说明，并明确后续验证与支付由用户手动完成。

## 高级动画语言

- 3D 景深：界面卡片在轻微透视空间中进入，配合柔和景深和视差移动。
- 玻璃展开：章节界面从半透明玻璃面板中逐层展开，边缘带克制的蓝紫光晕。
- 摄像机运动：使用缓入缓出的推近、横移和焦点切换，不使用突兀跳切。
- 光标反馈：点击时出现波纹、局部高光扫描和短促能量描边。
- 优先级重排：套餐卡片使用磁吸式位移动画，排名数字同步滚动。
- 刷新计数：数字使用纵向滚动、脉冲圆环和轻微粒子反馈表现真实递增。
- 自动点击：购买按钮进入聚焦锁定状态，描边汇聚后触发一次闪光，随后界面显示停止状态。
- 章节转场：使用流体渐变、动态遮罩和细线轨迹连接不同功能段落。
- 结尾动画：蓝紫霓虹线条汇聚成 GitHub 地址，背景粒子逐渐静止。
- 动效保持克制，字幕区域不被光效、粒子或界面卡片遮挡；避免频闪和剧烈抖动。

## 分镜

### 0–5 秒：标题

- 标题：`GLM Coding 订阅监测助手 Pro`
- 副标题：`多套餐优先级 · 自动刷新 · 单次点击`
- 粒子空间中由霓虹轨迹勾勒标题，玻璃界面从远处推近。

### 5–15 秒：套餐选择

- 展示插件总览。
- 聚焦月付、季付、年付筛选。
- 字幕：`自动读取不同订阅周期`
- 周期标签以高光扫描依次点亮，界面卡片带轻微 3D 视差。

### 15–25 秒：抢购优先级

- 勾选两个目标套餐。
- 展示上移、下移和移除控件。
- 字幕：`同时选择多个目标，从上到下依次尝试`
- 两个目标使用磁吸式重排，排名数字同步滚动。

### 25–35 秒：监测设置

- 聚焦 DOM 检测间隔、声音提醒、桌面通知。
- 字幕：`设置检测频率与本地提醒`
- 设置面板以分层玻璃卡片展开，开关带柔和脉冲反馈。

### 35–48 秒：运行状态

- 点击开始监测。
- 展示网页右上角状态卡，刷新次数递增。
- 字幕：`目标不可购买时刷新页面，并记录真实刷新次数`
- 刷新数字滚动增长，页面刷新通过动态遮罩和扫描线表现。

### 48–55 秒：自动点击逻辑

- 使用界面示意：购买按钮变为可用，插件自动点击一次并停止刷新。
- 字幕：`按钮可用时只点击一次，防止重复操作`
- 按钮进入聚焦锁定，描边汇聚后产生一次短促闪光。

### 55–60 秒：安全边界与链接

- 字幕：`验证码、登录、订单确认与支付仍需手动完成`
- 显示 GitHub：`github.com/zkilxx/glm-coding-watcher-pro`
- 霓虹轨迹汇聚成 GitHub 地址，粒子背景逐渐静止。

## 验证标准

- 成片分辨率为 1920×1080，时长 55–65 秒。
- 所有字幕均在安全区域内且无裁切。
- 音频无爆音，背景音乐不过度响亮。
- 动画保持 30fps 连贯，无明显卡顿、频闪或字幕遮挡。
- GitHub 地址清晰可读。
- 不包含个人信息、验证码细节、支付信息或误导性成功承诺。
- 抽取关键帧进行视觉检查，并完整播放验证 MP4 可解码。
