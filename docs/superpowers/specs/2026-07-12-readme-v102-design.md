# README v1.0.2 重写与截图设计

## 目标

以当前 v1.0.2 的真实功能重写仓库 README，删除所有视频和视频封面引用，只使用与当前界面一致的新截图说明安装、套餐读取、优先级、监测设置和页面状态。

## README 结构

1. 项目简介与限制声明。
2. 当前功能概览。
3. 新版功能截图。
4. 安装与更新，明确在 `chrome://extensions/` 核对版本号。
5. 读取月付、季付、年付套餐并设置多目标优先级。
6. 开始监测、立即刷新、单次点击和停止流程。
7. 悬浮窗状态说明：检查套餐中、抢购人数过多、暂时售罄、发现可购买套餐、已自动点击并停止刷新。

9. 本地验证与许可证。

README 不包含演示视频、视频封面、GIF 或指向视频文件的链接。

## 内容准确性

- 最低兜底检测间隔为 100ms；平衡模式 200ms，稳健模式 500ms。
- 冲刺模式最多 5 分钟。
- 未命中目标时立即刷新，无倒计时。
- 目标按名称、价格和订阅周期精确匹配。
- 购买按钮只点击一次，后续验证、订单确认和支付必须手动完成。
- 更新扩展后若现有页面脚本断联，读取套餐会自动重新注入并重试。
- 权限包含 `storage`、`notifications`、`tabs`、`scripting` 和限定的 BigModel 页面访问权限。
