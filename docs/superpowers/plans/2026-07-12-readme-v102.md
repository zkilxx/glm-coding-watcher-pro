## Global Constraints

- README 不包含视频、GIF、视频封面或视频链接。
- 截图不得包含账号、头像、验证码、支付信息或其他个人信息。
- 功能描述必须与 v1.0.2 代码和实际界面一致。
- 不启动监测、不点击购买、不处理验证码或支付。
